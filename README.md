## install

yarn install

nodeとyarnが必要です。

環境構築については下記URLを参照ください。

https://github.com/takahashim/js-stack-from-scratch

## task

* yarn start: webサーバーの起動とpugの監視
* yarn watch:css: scssの監視
* yarn watch:js: jsの監視
* yarn build: ビルド
* yarn build:dist: ビルド(フィンガープリントつき + サイトマップ出力)
